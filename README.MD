# ⚓ Lara Anchor

**Lara Anchor** is a lightweight, production-ready **Laravel Docker starter**.  
Spin up a full Laravel development and production stack with **PHP-FPM, Nginx, MySQL 8.3, Redis, Node**, all on **Alpine Linux**—fast, simple, and elegant.  

---

## 🚀 Features

- 🐳 **Dockerized Laravel stack** for development & production  
- ⚡ **Lightweight**: Alpine Linux base, minimal overhead  
- 🔌 **Full stack**: PHP-FPM, Nginx, MySQL 8.3, Redis, Node  
- 🛠️ **Customizable**: Modify Dockerfiles, memory limits, and services  
- 🌊 **Deep sea vibes**: aesthetic, modern setup inspired by the anchor theme
